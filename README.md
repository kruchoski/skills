# Skills

Claude Code skills for reasoning, writing, process mapping, and skill development.

Install any skill individually:

```bash
npx skills add kruchoski/skills --skill <skill-name>
```

---

## Reasoning & Analysis

| Skill | Description | Install |
|-------|-------------|---------|
| **epistemic-analysis** | Seven cognitive diversity agents analyze a question in parallel, then cross-examine disagreements to surface blind spots and strengthen decisions. | `npx skills add kruchoski/skills --skill epistemic-analysis` |
| **think-first** | Apply mental models (second-order effects, inversion, pre-mortem, and more) before major decisions. Enforces structured thinking before implementation. | `npx skills add kruchoski/skills --skill think-first` |
| **audience-gauntlet** | Run a proposal through 3-5 stakeholder personas in two passes: independent evaluation, then cross-pollination where personas react to each other's positions. | `npx skills add kruchoski/skills --skill audience-gauntlet` |
| **stakeholder-cabinet** | Simulate a deliberative panel of organizational roles (CTO, CFO, Legal, etc.) reacting to a proposal. Toggle between abstract roles or real people. | `npx skills add kruchoski/skills --skill stakeholder-cabinet` |
| **persona-simulator** | Create high-fidelity simulations of specific individuals in advisory, anticipatory, or collaborative modes. | `npx skills add kruchoski/skills --skill persona-simulator` |

## Writing & Voice

| Skill | Description | Install |
|-------|-------------|---------|
| **antislop** | Detect and fix AI-generated writing patterns. 35+ detection patterns across tiered severity, with editor mode for inline fixes. | `npx skills add kruchoski/skills --skill antislop` |
| **voice-analyzer** | Analyze writing samples to create a portable voice profile and style guide. Use when onboarding to voice-matched AI writing. | `npx skills add kruchoski/skills --skill voice-analyzer` |
| **voice-editor** | Edit AI-generated or draft content to match a voice profile. Three-pass editing: mechanical, register, precision. | `npx skills add kruchoski/skills --skill voice-editor` |
| **linkedin-draft** | Five-gate drafting workflow for LinkedIn posts: read source material, adversarial thesis review, draft with voice loaded, voice pre-flight, antislop scan. | `npx skills add kruchoski/skills --skill linkedin-draft` |
